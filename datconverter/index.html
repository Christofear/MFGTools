MFGTools DAT Converter

The professional bridge between NX library (.dat) files and Microsoft Excel.

MFGTools DAT Converter transforms the way you manage manufacturing data by moving complex .dat files into Excel, where you can use advanced logic, cross-referencing, and mass-editing tools to construct production-ready libraries in minutes rather than days.

The Power of Excel-Based Management

Instead of manual, line-by-line entry in text editors or standard interfaces, use the full power of spreadsheets:

Mass Library Construction

Quickly build entire databases by copying and pasting tabular data directly from vendor websites.

Intelligent Cross-Referencing

Connect different Excel sheets to manage relationships between various library components.

Logic-Driven Rules

Use Excel formulas to automate selections (e.g., If Diameter > 20mm, then assign Holder "CAT40_ER32").

Global Updates

Perform mass "Find & Replace" operations across thousands of records to update speeds, feeds, or material references.

Target Environment

Optimized for standard NX library structures:

...\MACH\resource\library\

Works with all standard ASCII databases:

  • Tools: tool_database.dat, shank_database.dat, trackpoint_database.dat, etc.
  • Holders: holder_database.dat
  • Assemblies: assembly_database.dat
  • Cut Parameters: cut_methods.dat, feeds_speeds.dat, machining_data.dat

Automation and Batch Processing

Full Command Line Interface (CLI) support. Use .bat or .cmd files to automate your entire library workflow:

# Example: Batch update all library files from a source folder
DatConverter.exe "C:\Library_Resources\Export" -o "C:\Library_Production" -y

Installation

  1. 1Download DatConverter.exe.
  2. 2Right-click the file → Properties → Check Unblock → OK.
  3. 3Launch the app and drag your .dat file onto the window.
